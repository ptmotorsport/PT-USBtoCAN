# RA4M1-Custom Arduino Board Package

## Overview

This repository provides the bootloader, GUI, and software for the PT Motorsport USB to CAN device.

## Features

- **Whitelist/Blacklist Filtering:** Filter for or against different CAN addresses
- **Adjustable CAN Speed** Choose between 125 kbps, 250 kbps, 500 kbps, and 1000 kbps
- **SAVVY CAN Compatibility** Can connect directly to Savvy CAN and/or generate readable files
- **LED indicators:** PWR, TX/RX, Error, and SD LEDs mapped for serial activity and debugging
- **SD Card Logging** Createst .txt files to store logged CAN messages
